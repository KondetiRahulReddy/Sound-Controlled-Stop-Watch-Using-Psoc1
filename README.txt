## Sound-Controlled-Stop-Watch-Using-Psoc1
##ESE 566-SBU Project
# This stop watch is built on PSOC1 and has 5 modes of operation -
a.) Accuracy State.
b.) Threshold State.
c.) Memory State.
d.) Microphone State.
e.) Button Press State.

The system works on a state machine and is implemented in C language.
